# Farmers Market Dashboard

This project is an interactive **Power BI dashboard** built to analyze sales activity in a farmers market.  
The goal of this dashboard is to understand vendor performance, product demand, and customer purchasing patterns to support better market decisions.

---

##  Project Overview

The dashboard analyzes sales data from a farmers market to identify trends in **product sales, vendor performance, and customer activity**.  

It helps answer questions like:

- Which products generate the most revenue?
- Which vendors contribute the highest sales?
- How are customers distributed across different locations?
- What products are most popular in the market?

---

##  Key Insights

- Total sales analyzed: **$8.67K**
- Total customers: **1,003**
- Vendors participating: **9**
- Booths in the market: **12**
- Top selling product: **Jalapeno Peppers ($3.19K in sales)**

These insights can help market organizers understand which products and vendors perform best and make better operational decisions.

---

##  Tools Used

- **Power BI**
- **DAX**
- **Power Query**
- **Microsoft Excel**

---

##  Dashboard Features

The dashboard includes:

- Sales performance overview
- Product-level revenue analysis
- Vendor performance comparison
- Customer distribution insights
- Interactive filters for:
  - Product
  - Vendor
  - Customer
  - Date

These filters allow users to explore the data and analyze trends easily.

---

##  How to Use

1. Download the `.pbix` file from this repository.
2. Open the file using **Power BI Desktop**.
3. Use the filters to explore different sales trends and insights.
